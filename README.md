# TODO_LIst
Todo List application written in kotlin on clean archicture

Component used:</br>
-MVVM</br>
-Room Database</br>
-Kotlin</br>
-Navigation Component</br>
-Data-binding</br>
-View-binding</br>
-Navigation fragment.</br></br>

Features:</br>
-Show todo by date.</br>
-Add date,priority to to do task.</br>
-Clean Ui.</br>
